 NVD3 Charts for Meteor
============================

This is the most recent Meteor package wrapping [nvd3](http://nvd3.org), the d3 charting library.  Based on a maintained [branch](http://liquidpele.github.io/nvd3/).

### Requirements
* d3

### Installation

    meteor add pfafman:nvd3
